一个稍带创新的俄罗斯方块
====================
四方向生成方块<br>
围成一整圈后消除<br>
下落速度递增<br>
得分随下落速度倍率增长<br>
善用 砸瓦鲁多！<br>
<br>
<br>
方向键 >控制上下左右<br>
空格键 >变形<br>
Z 字键 >时间减速<br>
回车键 >开始和死亡后重试<br>
<br>
//记得关闭输入法食用<br>

<br>

<br>
